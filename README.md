# Skills DZ — Landing Page

Page de landing pour Skills DZ, centre de formation professionnelle à Alger.

## Structure

```
skills-dz/
├── index.html
├── assets/
│   ├── css/styles.css
│   ├── js/main.js
│   └── img/
└── README.md
```

## Features

- Responsive (Mobile First)
- SEO optimized (meta tags, JSON-LD, OpenGraph)
- WhatsApp integration
- Form validation (Algerian phone format)
- Marketing tracking (GA4, GTM, Meta Pixel)
- WCAG 2.2 AA accessibility
- Animated counters
- FAQ accordion

## Configuration

Remplacez les placeholders :
- `+213XXXXXXXXX` → Numéro WhatsApp Skills DZ
- `GA4_ID` → ID Google Analytics
- `GTM_ID` → ID Google Tag Manager
- `PIXEL_ID` → ID Meta Pixel
- Image URLs → Vos propres images
